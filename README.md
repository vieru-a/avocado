## avocado

(FastAPI) Pet-project for a cross-platform application

## About

During the development of the project added:

  * settings for project
  * register
  * login/logout
  * verification
  * reset password
  * CRUD for users
  * script for creating superuser
    
## Technologies

- python 3.11
- FastAPI
- SQLAlchemy
- fastapi-users
- PostgreSQL
